SpaceInvader_AC::CAObject

Alias vers un tableau de 4 vecteurs de coordonnees.

  • indice 0 : les positions de l'envahisseur
  • indice 1 : les positions des missiles
  • indice 2 : les positions de mon vaisseau
  • indice 3 : les positions de mes torpilles
SpaceInvader_AC::CPosition

Alias vers une coordonnée de la matrice.

Permet d'activer la saisie à la volée.

Utile pour gérer les déplacement du joueur sans appuyer sur la touche entrée. si le joueur ne joue pas pendant 3 10ème de seconde, alors il pert une action.
